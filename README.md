# Personal Website

My personal website. Keeping it simple with a static deploy of a Next.js site using Github Pages and Cloudflare.

## Development

To run the development server:

```bash
npm run dev
```

## Building for Production

To create a production build:

```bash
npm run build
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- ESLint
